# J041_김병훈

### 선정한 퀘스트 : 2: 알고리즘 미식가 - AI의 창의적 비유

### 선정 이유 : 

평소 요리에 관심이 많은데 내가 배운 cs지식을 요리에 접목시킨다는 점에서 흥미로웠다. 요리와 코딩을 합한 분야에서 AI의 관점이 궁금했기에 '2번 알고리즘 미식가' 퀘스트를 선정하였다.

### 결과:

[CS 지식 한 줄 요약] :
자바스크립트는 단일 스레드지만, 비동기와 Web API 덕분에 멀티 스레드처럼 동작하는 착시를 만든다.

오늘의 추천 메뉴 🍜: "비동기 라멘, 콜백 토핑 추가"
끓는 국물(메인 스레드) 위에, 면은 미리 삶아두고(비동기 작업 예약), 타이밍 맞춰 말아내는 정교한 기술이 필요합니다.

 - 면은 setTimeout, fetch, Promise로 예약되고

 - 국물은 싱글 스레드 이벤트 루프가 하나씩 요리합니다.

 - 콜백 큐에서 하나씩 면을 꺼내올 때, 제대로 익었는지 확인하며 서빙!

CS 연결점 🔗:

 - 자바스크립트의 싱글 스레드는 오직 하나지만,

 - 이벤트 루프 + 콜백 큐 + Web API (브라우저 혹은 libuv) 가 함께 멀티 스레드처럼 보이는 비동기성을 연출해.

 - Worker Threads를 사용하면 실제 다중 스레드 처리도 가능하지.

### 느낀점 :

요리도 사실 여러 작업이 동시에 진행되는 멀티스레드라고 볼 수 있다는 생각이 들었다. 이렇듯 컴퓨터에 적용되는 코딩도 우리 일상생활에서 보이는 활동들로 비유시켜서 이해하기 쉽게 만드는 작업도 좋을 것 같다.
